# 📝 To-Do Web App

## 📌 Objective
A simple To-Do web app where users can:
- Add tasks
- View tasks instantly
- Delete tasks
- Mark tasks as completed

## 🚀 Features
- Add tasks dynamically
- Mark tasks as completed
- Delete tasks easily

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- index.html
- style.css
- script.js

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
